# Guide rapide de Markdown

Voici un tableau des principales syntaxes Markdown utilisées dans ce carnet.

| Rendu visuel | Code Markdown |
|--------------|---------------|
| **Texte en gras** | `**Texte en gras**` |
| *Texte en italique* | `*Texte en italique*` |
| ***Gras + italique*** | `***Gras + italique***` |
| ~~Texte barré~~ | `~~Texte barré~~` |
| [Lien vers GitHub](https://github.com) | `[Lien vers GitHub](https://github.com)` |
| ![Image exemple](./img/exemple.jpg) | `![Image exemple](./img/exemple.jpg)` |
| # Titre H1 | `# Titre H1` |
| ## Titre H2 | `## Titre H2` |
| ### Titre H3 | `### Titre H3` |
| - Liste à puces | `- Liste à puces` |
| 1. Liste numérotée | `1. Liste numérotée` |
| > Citation | `> Citation` |
| `Code inline` | `` `Code inline` `` |
| Bloc de code : | ```` ```python\nprint("Hello")\n``` ```` |
| Séparateur horizontal | `---` |
